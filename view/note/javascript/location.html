			  			assign()
			  			새로운 문서를 오픈함
					
    
					
    
			  			 reload(force)
			  			현재문서를 다시 읽어들임(Reload) 브라우저의 "새로고침"을 누른것과 동일한 효과 [전달인자] force 전달인자를 생략하거나 false로 지정하면, 문서가 마지막으로 불려온 이후로 변경된 경우에만 전체페이지를 다시 불러옴. 반대로 true이면 항상 서버에서 문서를 다시 불러옴 
					
    
					
    
			  			replace()
			  			현재 URL을 지정한 URL로 바꿔줌. 더 자세하게 말하면 브라우저의 세션 히스토리 상에서 새 항목을 생성하지 않고 현재의 문서를 새 문서로 교체.
